# 📊 Student Performance Classification

**DecodeLabs AI Internship – Project 2**

**Developed by:** Aanchal Pathak

## 📌 Overview

This project uses a **Decision Tree Classifier** to predict whether a student will **Pass** or **Need Improvement** based on their **study hours** and **attendance**.

## 🛠️ Technologies Used

* Python
* Pandas
* Scikit-learn

## ✨ Features

* Creates a sample student dataset.
* Trains a Decision Tree classification model.
* Predicts student performance.
* Displays model accuracy.
